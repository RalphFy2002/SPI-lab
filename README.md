Ce laboratoire a pour but de nous initier à la création d’IP et à l’utilisation du protocole SPI
afin de l’intégrer dans le diagramme de bloc du Minized à l’aide de Vivado. Une fois cette
intégration finie, le bon fonctionnement de l’IP sera vérifié par un code en C développé
dans SDK. Finalement, les sorties seront physiquement visualisées à l’oscilloscope et
numériquement sur l’ILA afin de valider le signal transmis.
